# Documentação — COLETAFortal

Bem-vindo à documentação do projeto **COLETAFortal**. Esta pasta contém informações técnicas e operacionais para desenvolvedores e colaboradores.

Arquivos nesta pasta:

- `SETUP.md` — instruções de instalação e execução (Windows / PowerShell).
- `DATA.md` — descrição dos dados geoespaciais (`src/backend/`), formatos e procedimentos de atualização.
- `ARCHITECTURE.md` — visão geral da arquitetura e organização do código.
- `REPORT_SUMMARY.md` — resumo do relatório do projeto (PDF `Relatório de projeto - COLETAFortal.pdf`) e instruções para extrair trechos.
- `CONTRIBUTING.md` — guia de contribuição, branchs e padrões de commit.

Se precisar que eu extraia trechos específicos do PDF e os incorpore aqui (por exemplo: objetivos, metodologia, resultados, mapas), diga quais páginas ou tópicos deseja incluir e eu faço a extração/adição.
